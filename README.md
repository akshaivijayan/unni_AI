# Unni AI Notes

Generate expert study notes from any YouTube video. Paste a link, and the app fetches the transcript and asks Mistral AI to produce clean, structured notes.

## Features
- YouTube transcript fetch (English)
- Expert notes generated by Mistral AI
- Rich, readable notes UI
- Export to Word and PDF

## Tech
- Node.js (HTTP server)
- youtube-transcript
- Mistral AI (Chat Completions API)

## Local Setup
1. Install dependencies
```bash
npm install
```
2. Create your `.env`
```bash
copy .env.example .env
```
3. Add your Mistral key
```
MISTRAL_API_KEY=your_key_here
```
4. Run the server
```bash
npm start
```
5. Open `http://localhost:3000`

## Environment Variables
- `MISTRAL_API_KEY` (required)
- `MISTRAL_MODEL` (optional, default `mistral-small-latest`)

## Deploy to Render (GitHub)
1. Push this repo to GitHub.
2. In Render, create a new **Web Service** and connect your GitHub repo.
3. Use:
```text
Build Command: npm install
Start Command: npm start
```
4. Add environment variables in Render:
```text
MISTRAL_API_KEY=your_key_here
MISTRAL_MODEL=mistral-small-latest
```
5. Deploy.

Render injects a `PORT` environment variable at runtime; the server reads it automatically. citeturn0search6

### One-click Render
This repo includes a `render.yaml`, so you can also use Render Blueprint deploy:
- It sets the build/start commands, health check, and env var placeholders.
